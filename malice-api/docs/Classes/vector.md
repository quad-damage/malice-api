# vector
Basic object for storing 3-axis vectors.

## Functions
### Setters
``.new(x: float, y: float, z: float)``

Creates a new vector with the given X, Y, and Z values.

## Getters
``:x()`` Returns the X field.

``:y()`` Returns the Y field.

``:z()`` Returns the Z field.
